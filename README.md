## 🔍 随手搜，不打断

任何页面按下 `Alt + S`，搜索框就弹出来了——不用切标签，不用动鼠标。

## 🎮 玩法

| 操作 | 效果 |
|------|------|
| `Alt + S` | 唤起搜索框 |
| `cd gg` | 切换到 Google 搜索 |
| `* 关键词` | 在收藏夹里搜书签 |

- **壁纸随心换** — 塞一张自己喜欢的图，每次打开都是好心情
- **搜索引擎一键切** — 输 `cd` 加引擎缩写，想用哪个用哪个
- **书签不再难找** — `*` 开头，直接搜收藏栏
- **输入有提示** — 搜什么，边打边猜
